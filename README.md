## html code

```
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
  </head>
 <body>

  <span
    class="typewriter"
    ty-text='["What is Lorem Ipsum?", "Why do we use it?", "Where does it come from?"]'
    ty-typing-speed="400"
    ty-delay="100"
    ty-cursor-speed="100">
  </span>
 
  <script src="https://cdn.jsdelivr.net/gh/shubham110019/typingtest@latest/typingtest.min.js"></script>
  </body>
</html>
```

## how to change cursor color
```

<style>
  .ty-cursor{
  color:#FA8072;
  }
</style>
```
